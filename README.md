# Traceway — API Monitoring Platform · Nuxt 3 Starter

Landing page for a SaaS API Monitoring platform with **Features**, **About**, and **Pricing** sections.

Stack: **Nuxt 3 · Tailwind CSS · IBM Plex · TypeScript**

---

## Getting started

```bash
npm install
npm run dev        # http://localhost:3000
npm run generate   # SSG build → .output/public
npm run build      # SSR build
```
